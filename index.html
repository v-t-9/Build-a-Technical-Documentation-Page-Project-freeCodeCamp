Introduction

Created by Guido van Rossum, Python is one of the most popular programming languages.

It works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc), it similarities with English makes it easy to read and, it uses indentation to define scope.

Uses

Python can be used:

  • on a server to create web applications.
  • alongside software to create workflows.
  • to connect to database systems. It can also read and modify files.
  • to handle big data and perform complex mathematics.
  • for rapid prototyping, or for production-ready software development.
Prerequisites and installation

To start with is recommended to have an IDE (integrated development environment) like Visual Studio Code already installed.

You can download python from here: https://www.python.org/downloads/

Get started

To write your first python program open your IDE and create a new file with the name "helloworld.py". Then write:

print("Hello World!")

This should print the string in the console.

Variables

In python a variable is created when you assign a value to it:

b = "Pyhton"

Python allows you to assign values to multiple variables in one line:

x, y, z = 1, 20, 15
Data types

Python data types are:

  • Text: str
  • Numeric: int, float
  • Sequence: list, tuples
  • Mapping: dict
  • Set: set, frozenset
  • Boolean: bool
  • Binary: bytes, bytearray, memoryview
